# Rebuild Url Rewrite for magento 2

Magento 2 module to rebuild all product url rewrites

## Installation

```
$ composer require "danielsousa/danielsousa/magento2-module-urlrewrite":"^1.0"
```

## Usage

```bash
 bin/magento urlrewrite:rebuild:products
```

## Prerequisites

- PHP >= 7.0.*
- Magento >= 2.1.*
